# Tip Calculator App

A responsive Tip Calculator built with HTML, SCSS (with PostCSS), and TypeScript, designed as part of a Frontend Mentor challenge. This app allows users to calculate the tip amount per person and the total cost per person based on the bill, selected tip percentage, and number of people.

## Demo

[Live demo link:](https://wgalik.github.io/tip-calculator-app-main_TypeScript/)

## Features

- Calculate tip per person and total per person dynamically.
- Preset tip buttons (5%, 10%, 15%, 25%, 50%) and a custom tip input.
- Responsive layout for mobile and desktop.
- Reset button to clear all inputs.
- Input validation ensures values are positive and within reasonable ranges.

Technologies

- **HTML5** – semantic markup
- **SCSS** – modular styles using variables and media queries
- **PostCSS** – for CSS transformations and autoprefixing
- **TypeScript** – type-safe DOM interactions and calculations
- Frontend Mentor Challenge – project based on the official challenge

## Usage

1. Enter the bill amount.
2. Select a tip percentage using the buttons or enter a custom tip.
3. Enter the number of people sharing the bill.
4. The app dynamically displays:
   - Tip amount per person
   - Total amount per person
5. Press Reset to clear all inputs and start over.
